Preview Window

The preview window is displayed at program startup by default, and any time a Preview button is pressed, or when the Preview Tab is clicked in the notebook tab field below the program menu. When the program starts, and prior clicking the button, the Preview Window will display a splash screen image if configured to do so (see the Preferrences section).

A scrollbar may be present in the right side of the Preview Window, as illustrated above, if the content (pages) cannot be displayed in one screen. As with most programs, the mouse wheel (if equipped) can be used to scroll through the material instead of using the scrollbar.

Most music score pages consist of 2 staffs or rows of music. If a single part is previewed by clicking on the below the Parts listbox in the Arranger, only those images (pages) are read from the database and displayed in the Preview Window. In these cases the scrollbar is not likely to be visible.

When arrangement music scores are displayed by pressing the below the Arranger listbox, the scrollbar will likely be visible because multiple images are usually associated with a single musical arrangement.

The Preview Window is also used by the Import Wizard to display the image files found in a selected directory to help the user map the image files to the part names in the hymn. This is an advanced topic, and is treated in more detail in the Import Wizard section of this document.
