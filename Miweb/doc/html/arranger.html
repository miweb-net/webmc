Arrange Hymn Parts


Arranger Functions

The Arranger is where hymn parts (e.g. Verse 1, Verse 2, Chorus, etc.) can be included in or excluded from the generated presentation file. The Arranger contains 2 list boxes and 2 buttons between them to control what parts of the hymn will go in the presentation file.

The left listbox (Parts) contains all of the available parts of the hymn. The right listbox (Arranger) contains the parts in the sequence they will be displayed -- in musical terms, this is the arrangement. Notice in the screen shot at the left that the Arranger listbox contains 5 Chorus (Coro) rows. This means that the hymn is usually sung in the sequence shown: Verse 1, then the Chorus, Verse 2 and the Chorus again, etc. But this usual sequence can be altered in a number of ways. The following sectionso on this page describe how to manipulate the content in the two Arranger listboxes.

Remove Parts from an Arrangment (Hymn)

It may be desired by the hymn director/song leader at times, to lead only selected parts of a hymn. One example is to sing the Coro only at the end of the hymn. If the default (usual) way is used, the presentation will be generated with the Coro between each verse. To change the presentation file to show the Coro only at the end of the hymn, select each Coro row in the Arranger listbox, except for the one following Verso 5 in this example. The listbox selection should look like the image on the right side of this document. To remove the selected parts from the Arranger listbox, press the button. All that should remain in the right Arranger listbox in this example is: Versos 1 - 5, and Coro.

Add Parts to an Arrangement (Hymn)

Parts can also be added to the Arranger listbox by selecting one or more parts from the Parts listbox and clicking the button. Notice that added parts are always appended to the bottom of the Arranger list. The next paragraph explains how to place them in the desired order.

Move Parts within an Arrangement (Hymn)

Parts can be moved anywhere within the Arranger listbox by dragging and dropping rows within the Arranger listbox. To perform a move, first press and hold the mouse button down on one of the rows in the Arranger listbox. While holding the mouse button down, move the mouse up or down the list. When the mouse is hovering over the location you want the dragged item to be, release the mouse button. Perhaps the best way to learn this process is through practicing. A step-by-step practical medley example illustrates these steps.

Clear Arranger Lists

Either or both of the Arranger listboxes can be emptied of all content by pressing the button below each listbox. When the button below one of the listboxes is pressed, all of the rows in the list will be removed.

Part or Arrangement Preview

Both Part and Arranger listboxes have a below them for viewing selected information. When pressing button below the Parts listbox, the music score pages of the selected part(s) will be displayed in the Preview Window. If the below the Arranger listbox is pressed, all rows in the Arranger listbox -- the arrangement -- are displayed in the Preview Window. In either case, of course the Preview Window is given focus (switched to) and the appropriate pages are displayed there. The arranger can be brought back into view, by clicking the icon in the notebook tabs just below the program menu.

Play Sound Sample

The Parts listbox has a Sound Sampler or Audition button below it. When pressed, this button will start a player program that will play the hymn that is selected in the Parts listbox. We'll borrow the medley example to illustrate the selected hymn. The selected hymn in the Parts listbox is Hymn 101. The text of the selected hymn is in bold font, it will often have a border around it, and usually has a slightly different background color from all other hymns in the Parts listbox. These styling features can be changed, so there could be a certain amount of ambiguity as to what is selected in the Parts listbox if the theme has been customized. This is the primary reason for caution when editing one of the provided themes.
A final note on the sample music that is played: If the file type is MIDI, the song sample will exactly match the displayed music score. It will include the first verse and chorus in most cases. There are rare instances where the entire hymn is captured in MIDI format.

Generate Hymn Presentation File

The final step in creating a hymn presentation file is to press the button, located just below the Arranger listbox. Under most circumstances, pressing this button will generate the presentation file and switch the view to the Organizer Tab. The icon will appear at the end of this Organizer list. But if the same hymn has already been generated -- even with different parts in its arrangement -- it will have the same name. When this is the case, a dialog is presented to the user to allow the program to overwrite the existing file. Pressing the Yes button will overwrite the file. So if unsure, press No. Note that the Media Center never overwrites or deletes a hymn in its database, so if a hymn presentation file (generated by the Media Center) is overwritten or deleted, it can be generated again from the the database.

Creating Medley Arrangements

This final subtopic of the Arranger, is an advanced feature. In addition to being able to arrange the order of the verses, chorus, multiple endings, etc. of a single hymn, the Parts and Arranger listboxes work together in a similar way to create medleys from multiple hymns. The medley example illustrates the following steps. The steps to creating a medley:

  1. Select the first hymn.
  2. Move parts around in the Arranger as desire (see how). Do not remove the first hymn from the Parts listbox prior to the next step.
  3. Select the second hymn by double-clicking it in the Search listbox.
  4. Click on the parts in the Parts listbox to be added to the bottom of the Arranger listbox.
  5. Rearrange the order of parts contained in the Arranger as desired.
  6. (optional) Repeat steps 3-5 to add additional hymn parts to the arrangement.
  7. Click the button to generate the medley presentation file.
