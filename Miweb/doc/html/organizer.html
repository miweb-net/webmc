Organizer


Organizer Functions

The organizer displays all of the generated presentation files created by the Media Center as well as those copied to the presentations subfolders using a file manager or command line utility. When the Media Center is started, it reads the hymns, scriptures, and other subdirectories to load existing content into the Organizer. Each of these 3 distinct presentation file types is represented by a specific icon related to its type. These icons that represent them enable distinguishing the type of presentation file at a glance.

The three recognized types are:

  • Music files (hymns)
  • Bible text files (scriptures)
  • Other Presentation files (.pptx extension)

One special case involves the creation of hymn medleys -- 2 or more hymns combined to make one hymn. These icons are identified by the word 'Medley', followed by the hymn numbers that comprise the song. Also, if the mouse is hovered over the icon, the titles of the individual hymns combined to make the medley will be displayed in a small tool tip window near the icon and mouse pointer. See the medley example for an illustration of this effect.

Refresh the Presentation List

The Media Center presentation list can be updated at any time by pressing the Organizer tab, which is located above the Organizer window. This feature is useful when a presentation file has been copied or moved from one folder to another, provided the new location is in a presentations subfolder. The Media Center does not periodically reread the presentations subfolders to find new or copied presentation files. It does recognize when new hymn or scripture presentation files are generated by the Media Center, and it also scans the presentations subfolders on startup.

Create Hymn/Scripture Presentations

Hymn and Scripture presentations can be automatically generated and sent to the Organizer using the Media Center. Hymn presentations are generated by pressing the button in the Arranger. Bible scripture presentations generated when button with the same icon is pressed within the Scripture Editor. Of course, the Organizer can be manually viewed by pressing the tab (located above the Organizer) at any time.

Change Order of Presentations

The Organizer provides standard drag and drop capabilties for organizing presentations in a particular order. To move an icon "ahead" of another one in the Organizer sequence, click the left mouse button and keep holding it down while you drag the item on top of the presentation it should precede. Once the mouse button is released, the drag will be complete and the icon will visually appear at the location at which it was dragged. The presentation sequence can be arranged in any order desired. When the program exits, it will save the order of presentation files as they appear in the Organizer, unless the preferences have been configured to remove the files in folder(s) on exit. The program saves the order of the remaining files so that when the program is restarted, they will appear in the Organizer in the same order as before program exit.

Display a Presentation

A presentation file is displayed by double-clicking its icon, or selecting the icon (single-click) and pressing the <Enter> key on the keyboard. As soon as one of these two methods of displaying the presentation file is executed, Libre Impress is launched to display the contents of the file. All control of the pages being displayed to the screen are managed through Libre Impress. Note that MIMC is still running even while the presentation is being displayed on screen(s).

Delete/Remove a Presentation

The Media Center will delete files from the Organizer when the program exits, if configured to do so. However, it may be necessary to delete a file manually. To do so, select a presentation icon by single-clicking on it, and then press the delete button to delete the file and remove the icon from the Organizer. Since there is no undo feature, a confirmation dialog is presented to the user to provide a measure of escape prior to deleting any file.

If the button is pressed, the selected file is permanently deleted from the system, and the icon is removed from the Organizer. If the button is pressed, the delete operation is aborted, the presentation icon is retained and no files are deleted.
