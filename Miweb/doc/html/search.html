Search For Hymns




Hymn Book Collection

The hymn book dropdown selects a hymnal, which is actually a database that (usually) contains the hymns for a specific hymnal. However, an arbitrary compilation of hymns can be stored in a database. Creating a custom database of hymns is beyond the scope of this document. Please occasionally check back at the Media Center Home page for this upcoming document.

This dropdown box provides the method for selecting which database contents are being displayed in the listbox below it. The list of songs figure is at the right. If the desired hymn is not displayed in the list and the Title Filter box (above the Hymn Book Collection) is empty, the hymn is not contained in the selected database. It may be contained in another database. Simply select another database and search within the new list of hymns for the desired title. If there are characters in the Title Filter box, they should be deleted so that all available songs in the selected database will be displayed. The Hymns Search Functions describes more about advanced hymn title searching using the Title Filter box.

Clicking on a song title in the list of hymns will causes the song information to be displayed in the box below the list of songs. This information is referred to as the hymn meta-data. It contains the title, number, authors and dates, as well as copyright information for the selected hymn.

The search list only supports selection of a single hymn. However, multiple hymns can be combined in the Arranger. Double-clicking a hymn in the search list will cause the selected hymn to be added to the Arranger. See the Arranger Section to learn more about customizing the presentation of single hymns and creating medleys, among its features.

Hymns Search Functions

The Media Center provides a listbox for displaying the titles in the selected hymnal (see Hymn Book Dropdown, above. The purpose of the Media Center is to enable the displaying of hymns, scriptures and other presentation files to TVs, the Hymns Search Listbox is key to that process. When a hymn title in the listbox is double-clicked, the Media Center shows that title as selected in the listbox, and displays the hymn's related data called meta-data below the listbox, and also populates the Arranger listboxes with the image parts that comprise the hymn.

Hymn Filter

Some hymn books (databases) contain a large set of songs, making it difficult to find a particular hymn -- especially by title. In these cases, part of the hymn title can typed in the Title Filter, which is an entry box located just above the Hymn Book Collection. As characters are typed, the list of hymns displayed in the Search Listbox gets shorter because titles are being filtered out that don't match what was typed in the Title Filter box. Deleting characters from the Title Filter entry causes the list to increase as more titles will match the characters in the Filter.
