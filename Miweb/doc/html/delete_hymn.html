Delete Hymn


Edit > Delete Menu Item

The Media Center provides a means for deleting a hymn that was added to a database (book) that doesn't belong in the database. If you use this maintenance feature, please be careful as hymn deletions are permanent. This means that the deleted hymn will no longer be available for use in the Media Center.

Occassionally a hymn must be deleted from a database because of any number of reasons. The most common are associated with the Importer, which is the utility provided by the Media Center to insert a hymn into a database. Here are the most common reasons for deleting a hymn from a database:

  • Imported hymn into an incorrect database.
  • Information associated with a hymn was typed incorrectly.
  • Hymn page mapping (mapping images to verses) was improper.
  • A hymn's copyright does not allow the church to project it.

If any of these reasons apply, it is probably necessary to use the delete feature, but ordinarily you should avoid using this function. It is an advanced feature that is meant to be used by personnel using the Importer (another advanced feature of the Media Center).

Deleting a hymn is a simple operation:

  1. Navigate to the Search page by clicking on the icon just below the menu.
  2. Select the database using the Hymn Book Dropdown
  3. Find the song in the hymn listbox and click once on it to select it.
  4. Click on the Edit > Delete menu item.

At this step, a confirmation dialog will be displayed to ensure that you intend to permanently delete the hymn from the database.
