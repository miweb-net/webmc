Mountain Island Media Center

User Guide


Introduction

The Mountain Island Media Center (Media Center) was developed for the purpose of organizing and displaying digital media to projectors and/or television screens supporting computer connected inputs such as VGA, DVI, HDMI, etc. This user manual describes the operation of the program as well as maintaining the generated material produced by the program.

The user interface for the program was designed to facilitate a simple work flow for worship media display during services of the church. Primarily, hymns and scripture texts are displayed to supplement hymn book and Bible content. The typical work flow starts on the Search notebook tab, illustrated by the icon. A hymn is selected using the controls on the Search page, and the work flow procedes toward the right side of the Search page, until the hymn is arranged in the far right list box. Finally, the selected hymn content is generated and subsequently directed to display on media devices such as TVs, or projectors.

Plan to run the Media Center on your primary monitor. This is because the primary monitor is almost never broadcast to a projection screen or display TV, and presentations will automatically be displayed on a secondary screen. This scenario enables the Media Center to continue to be used while a presentation is being displayed. Practice with it, and you'll develop your own way of using the Media Center to its fullest potential in your environment.

One of the most common tasks is to select a hymn to display to a projector or TV. The first step is to select the book (or database) to which the song belongs. This is accomplished by using the Hymn Book Collection List shown in the illustration above. Once the hymn book (database) is selected, a hymn can be selected from the Hymn List located immediately below the dropdown box. When a hymn is double-clicked, 3 things happen simultaneously:

  1. The hymn meta-data is displayed in the box below the selected hymn, and
  2. The Song parts list is filled with the names of the songs (e.g. Verse 1, Verse 2, Chorus, etc.)
  3. If there is only one hymn in the parts list, the song parts will automatically be arranged in the Arranger list box in a common order, closely matching hymn book suggestions. The Arranger list box is located at the far right side of the Search page.

Usually the next step is to press the (Generate Presentation) button, which will generate a presentation file containing all of the song part images in the order they are listed in the Arranger list box. The program will automatically switch the notebook tab to display the Organizer. The image at left shows the Organizer after some hymn presentation files have been generated. Double-clicking a presentation file will start the LibreOffice Impress application, which will load the presentation file, and display it in full-screen mode. There are additional features of the Arranger that can be used to customize the generated hymn presentation file.

A concise summary of the normal work flow for displaying a hymn is:

  1. Press the icon to show the Search page.
  2. Select a database source from the Search Hymn Collections.
  3. Double-click a hymn in the Search listbox located just below the Search Hymn Collections.
  4. Arrange the parts of the song to contain only desired pages/parts in the presentation (show me how).
  5. Press the (Generate Presentation) button.
  6. Double-click the hymn icon in the Organizer to display on an attached device.
